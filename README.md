## Tính năng chính 

| Module                | Chức năng nổi bật                                                                                 | DevExpress Controls chính                     |
|-----------------------|---------------------------------------------------------------------------------------------------|------------------------------------------------|
| Quản lý Sản phẩm      | Thêm/sửa/xóa, tự động cập nhật tồn kho, lịch sử nhập–xuất, tìm kiếm nhanh, phân loại             | GridControl, TileView, SearchControl, LookUpEdit |
| Loại & Thương hiệu    | Quản lý loại sản phẩm, thương hiệu (quốc gia)                                                     | GridControl + Popup Form                       |
| Khách hàng            | Tìm kiếm theo SĐT, phân loại giới tính, lịch sử mua hàng                                          | GridControl, RadioGroup                        |
| Nhân viên & Tài khoản | Phân quyền chi tiết, tạo tài khoản đăng nhập cho nhân viên                                        | LookUpEdit + Popup quản lý tài khoản           |
| Nhà cung cấp          | Quản lý NCC, xem danh sách phiếu nhập                                                             | GridControl                                    |
| Nhập hàng             | Tạo phiếu nhập, chọn NCC, nhiều dòng chi tiết, tự động cập nhật tồn, in phiếu                    | Master–Detail Grid, RepositoryItemLookUpEdit   |
| Bán hàng (POS)        | Giao diện POS hiện đại, quét mã/tìm tên, tính tiền tự động, in hóa đơn ngay                        | RibbonForm, Popup tìm SP, XtraReport           |
| Đăng nhập & Phân quyền| Login form đẹp, phân quyền chi tiết (bán hàng, nhập hàng, báo cáo, quản trị…)                    | XtraForm Login + Ribbon ẩn/hiện theo quyền     |
| Báo cáo & Thống kê    | Doanh thu ngày/tháng/năm, sản phẩm bán chạy, lợi nhuận, top khách, tồn kho thấp, biểu đồ         | ChartControl, PivotGrid, XtraReport            |
| Dashboard             | Tổng quan ngay khi mở phần mềm: doanh thu hôm nay, số đơn, tồn kho, top bán chạy…                | DashboardViewer / ChartControl + GaugeControl  |
